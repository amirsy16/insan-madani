📋 1. Persiapan Template CSV

📊 Template CSV yang Tersedia:
  • Template Donatur: 7 kolom (nama, gender, nomor_hp, email, alamat, pekerjaan, aktif)
  • Template Fundraiser: 3 kolom (nama_fundraiser, nomor_hp, aktif)
  • Template Donasi: 5 kolom (donatur, jenisDonasi, metodePembayaran, jumlah, tanggal_donasi)
⚠️ Hal Penting Sebelum Mengisi Template:
  • Download template CSV yang sesuai dengan jenis data yang akan diisi
  • Gunakan aplikasi seperti Excel, Google Sheets, atau LibreOffice Calc
  • Pastikan menyimpan file dalam format CSV dengan encoding UTF-8
  • Jangan mengubah nama kolom (header) pada baris pertama
  • Ikuti format data sesuai contoh yang diberikan
  • Validasi semua data sebelum menyimpan file CSV
💡

Tips Persiapan Template:

  1. Download Template: Ambil template CSV kosong dari sistem
  2. Siapkan Data: Kumpulkan semua data yang akan dimasukkan
  3. Cek Format: Pastikan format data sesuai dengan kolom yang tersedia
  4. Test Data: Isi 2-3 baris contoh untuk testing
  5. Backup: Simpan copy data asli sebelum diubah ke format CSV
  • Pastikan semua data wajib sudah diisi
  • Import data dalam urutan: Donatur → Fundraiser → Donasi
  • Backup data sebelum import

    📋 Struktur Template Donatur (7 kolom)

    Langkah Import Donatur:

    1. Buka halaman Donatur di menu navigasi
    2. Klik tombol "Import" di bagian atas tabel
    3. Upload file CSV yang sudah disiapkan
    4. Review data preview dan klik "Import"
    5. Tunggu proses selesai dan cek hasilnya

    💰 4. Import Data Donasi

    ⚠️ Pastikan data Donatur dan Fundraiser sudah diimport terlebih dahulu!

    Kolom Template Donasi (5 kolom)

    Kolom Wajib Format Contoh Keterangan
    donatur Ya Kode/Email/HP DN25001 Kode donatur, email, atau nomor HP
    jenisDonasi Ya Text Zakat Maal Sesuai master data jenis donasi
    metodePembayaran Tidak Text Transfer Bank Sesuai master data metode pembayaran
    jumlah Ya Number 500000 Jumlah dalam rupiah (tanpa titik)
    tanggal_donasi Ya YYYY-MM-DD 2025-06-15 Format tanggal ISO

    📋 Master Data Reference

    Jenis Donasi yang Tersedia:

    • Zakat Maal • Zakat Fitrah • Zakat Perusahaan • Infaq Terikat • Infaq Tidak Terikat • Sedekah • Dana CSR • DSKL (Dana Sosial Keagamaan Lainnya) • Donasi Logistik/Barang

    Metode Pembayaran yang Tersedia:

    • QRIS (Umum) • LinkAja • GoPay • OVO • DANA • Tunai (Cash) • Transfer Bank

    📝 Contoh Template CSV Donasi:

    donatur,jenisDonasi,metodePembayaran,jumlah,tanggal_donasi DN25001,Zakat Maal,Transfer Bank,500000,2025-06-15 81234567890,Infaq Tidak Terikat,QRIS,100000,2025-06-15 ahmad.budi@gmail.com,Sedekah,Tunai,50000,2025-06-14 DN25002,Zakat Fitrah,GoPay,25000,2025-06-13
    3

    Langkah Import Donasi:

    1. Pastikan data Donatur dan Fundraiser sudah ada
    2. Buka halaman Donasi di menu navigasi
    3. Klik tombol "Import" di bagian atas tabel
    4. Upload file CSV yang sudah disiapkan
    5. Review data preview dengan teliti
    6. Klik "Import" dan tunggu proses selesai

    🔧 5. Troubleshooting & Tips

    ❌ Error yang Sering Terjadi

    1. Format Tanggal Salah

    ❌ Salah: 15/06/2025, 15-06-2025, 2025/06/15
    ✅ Benar: 2025-06-15

    2. Format Nomor HP Salah

    ❌ Salah: 081234567890, +6281234567890, 08123-456-7890, 6281234567890
    ✅ Benar: 81234567890

    3. Nama Jenis Donasi Tidak Sesuai

    ❌ Salah: Zakat, Infaq, Sodakoh
    ✅ Benar: Zakat Maal, Infaq Tidak Terikat, Sedekah

    4. Donatur Tidak Ditemukan

    Solusi: Pastikan kode donatur, email, atau nomor HP sudah ada di sistem sebelum import donasi.

    💡 Tips Sukses Import

    ✅ Best Practices:

    • Test dengan data kecil - Import 2-3 baris dulu untuk test
    • Cek encoding - Pastikan file CSV menggunakan UTF-8
    • Hindari karakter khusus - Gunakan tanda kutip untuk teks yang mengandung koma
    • Backup terlebih dahulu - Selalu backup data sebelum import besar
    • Import bertahap - Jangan import ribuan data sekaligus
    • Validasi manual - Cek beberapa data hasil import secara manual

    📊 Format Excel ke CSV

    💾

    Cara Konversi Excel ke CSV UTF-8:

    1. Buka file Excel Anda
    2. Klik File → Save As
    3. Pilih format "CSV UTF-8 (Comma delimited)"
    4. Beri nama file dan klik Save
    5. Jika ada peringatan, klik Yes untuk melanjutkan

    🔍 Validasi Data Sebelum Import

    Checklist Sebelum Import:

    • ☑️ Semua kolom wajib sudah terisi
    • ☑️ Format tanggal: YYYY-MM-DD
    • ☑️ Format nomor HP: 8xxxxxxxxx
    • ☑️ Email format valid (ada @ dan domain)
    • ☑️ Jenis donasi sesuai master data
    • ☑️ Metode pembayaran sesuai master data
    • ☑️ Jumlah donasi adalah angka (tanpa titik pemisah)
    • ☑️ Gender: male atau female
    • ☑️ Status aktif: true atau false

    📞 Support & Bantuan

    Jika mengalami kesulitan:

    • Periksa kembali format data sesuai panduan ini
    • Test dengan data sedikit terlebih dahulu
    • Pastikan semua master data (jenis donasi, metode pembayaran) sudah tersedia
    • Hubungi administrator sistem untuk bantuan lebih lanjut
