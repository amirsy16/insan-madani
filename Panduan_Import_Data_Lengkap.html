📋 1. Overview & Persiapan Template

✨ Template CSV yang Tersedia:
  • Donatur: 6 kolom utama (nama, gender, nomor_hp, email, alamat, pekerjaan_id)
  • Fundraiser: 3 kolom utama (nama_fundraiser, nomor_hp, aktif)
  • Donasi: 5 kolom utama (donatur, jenisDonasi, metodePembayaran, jumlah, tanggal_donasi)
⚠️ Penting Sebelum Mengisi Template:
  • Gunakan format CSV dengan encoding UTF-8
  • Pastikan semua data wajib sudah diisi
  • Isi template dalam urutan: Donatur → Fundraiser → Donasi
  • Validasi data sebelum menyimpan sebagai CSV

👤 2. Pengisian Template Donatur

Kolom Template Donatur (6 kolom)

Kolom Wajib Format Contoh Keterangan
nama Ya Text Ahmad Budi Santoso Nama lengkap donatur
gender Ya male/female male Jenis kelamin (male atau female)
nomor_hp Tidak 8xxxxxxxxx 81234567890 Format nomor HP tanpa awalan
email Tidak email@domain.com ahmad.budi@gmail.com Email valid
alamat Tidak Text Jl. Merdeka No. 123, RT 02/RW 05 Alamat lengkap
pekerjaan_id Tidak Text (Sesuai Master Data) Karyawan Swasta Nama pekerjaan sesuai master data

Langkah Mengisi Template Donatur:

  1. Download template kosong: Dapatkan file template_import_donatur.csv
  2. Buka dengan Excel/Google Sheets: Pastikan dapat melihat 6 kolom header
  3. Isi data baris per baris: Mulai dari baris kedua (setelah header)
  4. Validasi setiap kolom: Pastikan format sesuai dengan panduan
  5. Simpan sebagai CSV UTF-8: Gunakan format yang tepat
  6. Test dengan data kecil: Coba 2-3 baris dulu sebelum data lengkap

⚠️ Validasi Khusus Template Donatur:

  • Kolom "nama": Wajib diisi, minimal 3 karakter
  • Kolom "gender": Hanya boleh "male" atau "female" (huruf kecil)
  • Kolom "nomor_hp": Format 8xxxxxxxxx (dimulai 8, tanpa prefix 62 atau 0)
  • Kolom "email": Harus format email valid dengan @ dan domain
  • Kolom "pekerjaan_id": Harus sesuai dengan master data pekerjaan yang tersedia

📖 Panduan Detail Pengisian Setiap Kolom Donatur:

📝

Petunjuk Kolom per Kolom:

1. Kolom "nama" (WAJIB)
  • Isi dengan nama lengkap donatur
  • Minimal 3 karakter, maksimal 255 karakter
  • Boleh menggunakan spasi dan tanda baca biasa
  • Contoh: "Ahmad Budi Santoso", "Dr. Siti Nurhaliza, S.Pd"
2. Kolom "gender" (WAJIB)
  • Hanya boleh diisi: male atau female
  • Huruf kecil semua, tidak boleh ada spasi
  • male = Laki-laki, female = Perempuan
3. Kolom "nomor_hp" (OPSIONAL)
  • Format: 8xxxxxxxxx (dimulai 8, tanpa prefix 62 atau 0)
  • Contoh: 81234567890, 87654321098
  • Jika tidak ada, kosongkan saja
4. Kolom "email" (OPSIONAL)
  • Harus format email yang valid
  • Contoh: ahmad.budi@gmail.com, siti@yahoo.co.id
  • Jika tidak ada, kosongkan saja
5. Kolom "alamat" (OPSIONAL)
  • Isi alamat lengkap donatur
  • Boleh menggunakan koma, titik, dan karakter khusus
  • Contoh: "Jl. Merdeka No. 123, RT 02/RW 05, Kelurahan ABC"
6. Kolom "pekerjaan_id" (OPSIONAL)
  • Harus sesuai dengan master data pekerjaan yang tersedia
  • Pilihan yang tersedia: Belum/Tidak Bekerja, Pelajar/Mahasiswa, Mengurus Rumah Tangga, Pensiunan, Buruh, Dokter, Dosen, Guru, Karyawan Swasta, Nelayan, Pedagang, Pegawai Negeri Sipil, Petani, POLRI, TNI, Wiraswasta, Lainnya
  • Nama harus persis sama dengan yang ada di sistem (perhatikan huruf besar/kecil)
  • Jika tidak sesuai, biarkan kosong atau pilih "Lainnya"

🎯 3. Pengisian Template Fundraiser

📋 Struktur Template Fundraiser (3 kolom)

Kolom Wajib Format Contoh Keterangan
nama_fundraiser Ya Text Ahmad Fundraiser Nama lengkap fundraiser
nomor_hp Tidak 8xxxxxxxxx 81234567890 Format nomor HP tanpa awalan
aktif Ya true/false true Status aktif fundraiser

📝 Contoh Pengisian Template CSV Fundraiser:

nama_fundraiser nomor_hp aktif
Ahmad Fundraiser 81234567890 true
Budi Pengumpul 87654321098 true
Siti Koordinator 85555444333 false
2

Langkah Mengisi Template Fundraiser:

  1. Download template kosong: Dapatkan file template_import_fundraiser.csv
  2. Buka dengan aplikasi spreadsheet: Pastikan dapat melihat 3 kolom header
  3. Isi data fundraiser: Mulai dari baris kedua setelah header
  4. Periksa nomor HP: Pastikan format 8xxxxxxxxx
  5. Set status aktif: Gunakan "true" untuk aktif, "false" untuk non-aktif
  6. Simpan sebagai CSV UTF-8: Pastikan encoding benar

💡 Tips Mengisi Template Fundraiser:

  • Nama Fundraiser: Isi nama lengkap yang mudah dikenali
  • Nomor HP: Opsional, tapi sangat disarankan untuk komunikasi
  • Status Aktif: Set "false" jika fundraiser sudah tidak aktif
  • Template ini paling sederhana karena hanya 3 kolom

💰 4. Pengisian Template Donasi

⚠️ Pastikan template Donatur dan Fundraiser sudah diisi dan diimport terlebih dahulu!

Kolom Template Donasi (5 kolom)

Kolom Wajib Format Contoh Keterangan
donatur Ya Kode/Email/HP DN25001 Kode donatur, email, atau nomor HP
jenisDonasi Ya Text Zakat Maal Sesuai master data jenis donasi
metodePembayaran Tidak Text Rekening Zakat BSI 241... Sesuai master data metode pembayaran
jumlah Ya Number 500000 Jumlah dalam rupiah (tanpa titik)
tanggal_donasi Ya YYYY-MM-DD 2025-06-15 Format tanggal ISO

📋 Master Data Reference

Jenis Donasi yang Tersedia:

• Zakat Maal • Zakat Fitrah • Zakat Perusahaan • Infaq Terikat • Infaq Tidak Terikat • Sedekah • Dana CSR • DSKL (Dana Sosial Keagamaan Lainnya) • Donasi Logistik/Barang

Metode Pembayaran yang Tersedia:

• QRIS (Umum) • LinkAja • GoPay • OVO • DANA • Tunai (Cash) • Rekening Zakat BSI 241... • Rekening Zakat BSI 103... • Rekening Zakat Muamalat 441... • Rekening Infak BSI 103... • Rekening Infak Mandiri 110...

💡 Panduan Pemilihan Metode Pembayaran:

  • Digital Payment: QRIS, LinkAja, GoPay, OVO, DANA
  • Transfer Bank Zakat: Gunakan rekening khusus zakat (BSI 241..., BSI 103..., Muamalat 441...)
  • Transfer Bank Infak: Gunakan rekening khusus infak (BSI 103..., Mandiri 110...)
  • Tunai: Untuk donasi yang diterima secara langsung
  • Nama harus persis sama dengan yang ada di master data sistem

81234567890 Infaq Tidak Terikat QRIS (Umum) 100000 2025-06-15
ahmad.budi@gmail.com Sedekah Tunai (Cash) 50000 2025-06-14
DN25002 Zakat Fitrah GoPay 25000 2025-06-13

Langkah Mengisi Template Donasi:

  1. Download template kosong: Dapatkan file template_import_donasi.csv
  2. Buka dengan aplikasi spreadsheet: Pastikan dapat melihat 5 kolom header
  3. Pastikan referensi donatur: Gunakan kode donatur, email, atau nomor HP yang valid
  4. Pilih jenis donasi: Sesuaikan dengan master data yang tersedia
  5. Tentukan metode pembayaran: Pilih sesuai dengan cara pembayaran sebenarnya
  6. Isi jumlah dan tanggal: Pastikan format angka dan tanggal benar
  7. Simpan sebagai CSV UTF-8: Pastikan encoding dan format benar

🔧 5. Troubleshooting & Tips

❌ Error yang Sering Terjadi

1. Format Tanggal Salah

❌ Salah: 15/06/2025, 15-06-2025, 2025/06/15
✅ Benar: 2025-06-15

2. Format Nomor HP Salah

❌ Salah: 081234567890, +6281234567890, 08123-456-7890, 6281234567890
✅ Benar: 81234567890

3. Nama Jenis Donasi Tidak Sesuai

❌ Salah: Zakat, Infaq, Sodakoh
✅ Benar: Zakat Maal, Infaq Tidak Terikat, Sedekah

4. Donatur Tidak Ditemukan

Solusi: Pastikan kode donatur, email, atau nomor HP sudah ada di sistem sebelum import donasi.

💡 Tips Sukses Import

✅ Best Practices:

  • Test dengan data kecil - Import 2-3 baris dulu untuk test
  • Cek encoding - Pastikan file CSV menggunakan UTF-8
  • Hindari karakter khusus - Gunakan tanda kutip untuk teks yang mengandung koma
  • Backup terlebih dahulu - Selalu backup data sebelum import besar
  • Import bertahap - Jangan import ribuan data sekaligus
  • Validasi manual - Cek beberapa data hasil import secara manual

📊 Format Excel ke CSV

💾

Cara Konversi Excel ke CSV UTF-8:

  1. Buka file Excel Anda
  2. Klik File → Save As
  3. Pilih format "CSV UTF-8 (Comma delimited)"
  4. Beri nama file dan klik Save
  5. Jika ada peringatan, klik Yes untuk melanjutkan

🔍 Validasi Data Sebelum Import

Checklist Sebelum Import:

  • ☑️ Semua kolom wajib sudah terisi
  • ☑️ Format tanggal: YYYY-MM-DD
  • ☑️ Format nomor HP: 8xxxxxxxxx
  • ☑️ Email format valid (ada @ dan domain)
  • ☑️ Jenis donasi sesuai master data
  • ☑️ Metode pembayaran sesuai master data
  • ☑️ Jumlah donasi adalah angka (tanpa titik pemisah)
  • ☑️ Gender: male atau female
  • ☑️ Status aktif fundraiser: true atau false
  • ☑️ Pekerjaan donatur sesuai master data

📞 Support & Bantuan

Jika mengalami kesulitan:

  • Periksa kembali format data sesuai panduan ini
  • Test dengan data sedikit terlebih dahulu
  • Pastikan semua master data (jenis donasi, metode pembayaran) sudah tersedia
  • Hubungi administrator sistem untuk bantuan lebih lanjut
